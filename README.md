# Yandex music desktop app
Yandex Music web app. Electron powered. Supports macOS, Windows, Linux.

Support media keys (play/pause/prev/next)

# Build Requirements

* https://github.com/alexandrius/yandex.music

# Build

```bash
$ ./build.sh
```
